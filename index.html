クラウド同期（複数端末でデータ共有）

スマホ・タブレット・PCで同じデータ（案件・写真・日報すべて）を共有できます。データは自分のサーバー／自分のGitHub非公開リポジトリにだけ保存され、外部業者には渡りません。

Xサーバー方式の初回設定（1回だけ）

① 「genba-api」フォルダ一式を、Xサーバーのファイルマネージャで public_html の中にアップロード（ZIPをアップして展開でOK）
② 上の「APIのURL」に https://あなたのID.xsrv.jp/genba-api/api.php を入力
③ 「合言葉」に、サーバーの config.php に書かれている合言葉を貼り付け
④ 「保存して今すぐ同期」を押す
⑤ 他の端末でも同じURLと合言葉を入れるだけで、同じデータが使えます
※ 合言葉はこの端末の中にだけ保存されます。他人に教えないでください。
※ サーバー側には自動で直近10世代のバックアップが残ります。

GitHub方式の初回設定（トークンの作り方・1回だけ）

① GitHubにログインし Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token を開く
② Repository access：「Only select repositories」→ genba-report-data を選ぶ
③ Permissions → Repository permissions → Contents を Read and write にする
④ 有効期限を設定して作成し、表示されたトークンを上の欄に貼り付けて「保存して今すぐ同期」
⑤ 他の端末でも設定画面に同じトークンを貼れば、同じデータが使えます
※ トークンはこの端末の中にだけ保存されます。他人に教えないでください。

プッシュ通知

連絡板に新しいメッセージが届いたとき、この端末に通知します（送った本人には通知されません）。端末ごとにオンにしてください。

iPhone・iPad：「ホーム画面に追加」したアイコンから開いた場合のみ通知を使えます（iOS 16.4以降）。Safariで直接開いている間はボタンが反応しません。

スマホ・タブレットで使う

このアプリはホーム画面に追加すると、普通のアプリと同じように全画面・オフラインで使えます（インストール不要・無料）。

iPhone・iPad：Safariで開く → 共有ボタン（□↑）→「ホーム画面に追加」
Android：Chromeで開く → 右上メニュー（⋮）→「ホーム画面に追加」
※ ホーム画面から起動すれば、電波のない現場でもそのまま使えます。

データ管理

すべてのデータはこの端末内（ブラウザ）にのみ保存されます。機種変更前・定期的にエクスポートしてください。事務所のPCでインポートすれば、現場の状況をそのまま確認できます。
